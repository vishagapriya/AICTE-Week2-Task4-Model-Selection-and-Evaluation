# Week 2 - Task 4: Model Selection and Evaluation

In this task, I learned how different machine learning models perform on the same dataset and how to select the most suitable model using evaluation metrics.

Models Used:

* Logistic Regression
* Decision Tree Classifier
* Random Forest Classifier

Steps Performed:

1. Loaded and cleaned the dataset.
2. Split the data into training and testing sets.
3. Trained multiple machine learning models.
4. Evaluated the models using accuracy, precision, recall and F1-score.
5. Applied cross-validation to get more reliable results.
6. Compared the performance of all models.

Files Included:

* model_selection_evaluation.py
* model_comparison_results.csv
* README.md

Conclusion:
After comparing the models, Random Forest showed better overall performance and consistency. This task helped me understand how model evaluation is important before selecting a final machine learning model.
